### Нумерованный список
```
1. Пункт первый
2. Пункт второй
3. Пункт третий
```
***Пример:***

1. Пункт первый
2. Пункт второй
3. Пункт третий
---

### Маркированный список
```
- Пункт первый
- Пункт второй
- Пункт третий
```
***Пример:***

- Пункт первый
- Пункт второй
- Пункт третий
---
### Вложенные списки
Также можно делать вложенные списки, добавляя 4 пробела перед пунктом:
```
1. Пункт первый
    - Подпункт первый
    - Подпункт второй
2. Пункт второй
```
***Пример:***

1. Пункт первый
    - Подпункт первый
    - Подпункт второй
2. Пункт второй
