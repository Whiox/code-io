## Блоки кода
### Строка кода
```
`строка кода`
```
***Пример:***

`строка кода`

---
### Блок кода

Удалите символы `\`
```
\```
Блок кода
\```
```
***Пример:***

```
Блок кода
```

---
### Подсветка кода
Для блоков кода можно указывать язык программирования.

Используется подсветка синтаксиса из библиотеки `linguist`, которая включает множество различных языков.

Удалите символы `\`
```
\```python
print("Привет, мир!")
\```
```
***Пример:***

```python
print("Привет, мир!")
```
